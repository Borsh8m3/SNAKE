# SNAKE
A simple snake console game, in C++, developed during classes at university.
# Running the game locally
1. Clone the repository or download all files below.
2. Sort the files into appropriate folders

Source files:
- main.cpp
- Menu.cpp
- Plansza.cpp

Headre files:
- Menu.h
- Obiekt.h
- Plansza.h
- Polozenie.h
- Waz.h

Enjoy the game :)
